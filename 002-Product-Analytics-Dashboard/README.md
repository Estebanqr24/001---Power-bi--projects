# 📊 Retail Dashboard

Dashboard desarrollado en Power BI para analizar las ventas de de un retail, así como sus ingresos, por medio de periodos de tiempo, por ejemplo, de manera mensual, anual, trimestral. 

---

## 🛠️ Herramientas utilizadas
- Power BI
- DAX
- Excel para leer archivos csv y xlsx
- Ms Access

---

## 📈 KPIs y análisis
- Ventas totales
- Ingresos totales
- Unidades vendidas por periodo de tiempo
- Títulos dinámicos realizados con DAX


---

## 🧠 Funciones DAX utilizadas
- Format
- LastBlank
- Counta
- Calculate
- PREVIOUSYEAR
- DATESYTD
- RELATED
- CONCATENATE
- DIVIDE
- SAMEPERIODLASTYEAR
- SUM

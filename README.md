# 📊 Power BI Projects
Repositorio orientado al desarrollo de proyectos de análisis y visualización de datos utilizando Power BI, DAX y distintas fuentes de información.
Aquí se agrupan dashboards, reportes, KPIs, gráficos y análisis comerciales desarrollados como parte de mi proceso de aprendizaje y construcción de portafolio en Business Intelligence y análisis de datos.
---
## 📁 Estructura del repositorio
Cada proyecto se encuentra organizado en carpetas consecutivas con el siguiente orden:
```text
001-sales-performance
002-<nombre-proyecto>
003-<nombre-proyecto>
...
Dentro de cada carpeta de proyecto se mantiene una estructura estandarizada:
001-sales-performance/
│
├── data/                  # Fuentes de datos (Excel, CSV, SQL, etc.)
├── powerbi/               # Archivo .pbix del tablero de Power BI
├── screenshots/           # Capturas de pantalla del dashboard
└── README.md              # Descripción detallada del proyecto
